Action and Event Tagging

Action and Event Tagging aims to recognize the actions and events from a match footage reducing time spent for digital broadcasting/ video production companies. Player Tracker

SportsVision’s Player Tracker is a state-of-the-art tracker algorithm to track players, predefined objects and custom objects in a match footage. It can detect the speed of players, track flying trajectory, action areas inside a match, and many more such parameters to improve the performances.
